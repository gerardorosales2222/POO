# 📚 Trabajos Prácticos y Actividades - POO

Esta carpeta contiene el material práctico y los desafíos planteados para la materia. El objetivo es aplicar los pilares de la Programación Orientada a Objetos mediante la resolución de problemas reales.

## Listado de Actividades

A continuación, encontrarás los enlaces a las consignas y el código base de cada trabajo. Si bien están las soluciones en las carpetas, se sugiere resolver por cuenta propia y luego corroborar con las soluciones brindadas por la cátedra:

* **[Actividad 01: Billetera Virtual](https://github.com/gerardorosales2222/POO/tree/main/00%20Trabajos/01%20Actividad01%20-%20BilleteraVirtual)** Modelado inicial de clases y gestión de saldos.

* **[Actividad 02: Juego de Personajes](https://github.com/gerardorosales2222/POO/tree/main/00%20Trabajos/02%20Actividad02%20-%20JuegoPersonajes)** Implementación de lógica de combate, constructores y métodos.

* **[Actividad 03: Sistema de Mensajes](https://github.com/gerardorosales2222/POO/tree/main/00%20Trabajos/03%20Actividad03%20-%20Mensajes)** Práctica de comunicación entre objetos y flujo de datos.

## Requisitos
* Java JDK 8 o superior.
* Un IDE o editor de texto (NetBeans, IntelliJ, Eclipse).
* Ganas de programar.
